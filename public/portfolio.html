  • Compilers

    • exp: a programming langauge and compiler

      exp is a basic c inspired programming langauge of my own design, which I wrote the compiler,
      'exp', for. I created this langauge and compiler in order to have a place to learn about how code generation works. As I have written several 'toy' compilers before targeting LLVM or simply interpreting the language
      I think the most impressive thing about the project is that I wrote it in C, with only the standard library.
      You are welcome to take a look if you like.

    • tbc: tiny bytecode compiler

      tbc, (or Tiny Bytecode Compiler), is a project to allow something like exp to run on the web.
      where I can show it to people easily and they can interact with it. it has the same basic structure
      as exp, as a bytecode compiler. However, instead of a translation pass into x86_64 assembly it only
      has a bytecode interpreter. The new part of this is that we can compile the bytecode interpreter
      to web assembly and run it in the browser which is exactly what I did!
