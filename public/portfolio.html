exp

exp is a basic c inspired programming langauge of my own design,
which I wrote the compiler, 'exp', for.
I created this langauge and compiler in order to have a place to learn about how compilers work.
The langauge itself is minimalistic to say the least, it only supports one integer type,
function definitions and function calls, as well as a tuple type which is it's most
recent addition. You are welcome to take a look if you like.
Though it's probably fairly amateurish in terms of design and implementation.
I think the most impressive thing about the project is that I wrote it in C, with no libraries.

tbc

tbc, the Tiny Bytecode Compiler, is a project to allow something like exp to run on the web
where i can show it off to other easily. it has the same basic structure as exp as a bytecode compiler
However, instead of a translation pass into x86_64 assembly it only has a bytecode interpreter.
The new part of this is that we can easily compile the bytecode interpreter to web assembly and run it in the browser,
where you can play with it!
